# 퍼펙토익 게임화 기획서 — "레벨업 저니 + 스피드 스네이크"

> 목적: 학습이 지루하다는 문제를 **"레벨업 → 보상 → 또 하고 싶음"** 이라는 게임의 핵심 루프로 해결한다.
> 구현 진행은 CLAUDE.md 세션 로그에 기록. 이 문서는 설계 단일 진실원.

## 0. 설계 원칙
토익 학습(정독·판단)과 아케이드 액션은 충돌하므로 **"성장 시스템"과 "액션 모드"를 분리**하고 둘을 **XP** 하나로 잇는다.
- 성장 시스템(XP·레벨·티어·보상)은 전 콘텐츠(연습·대결·리스닝·몸풀기·진단·스네이크)에 얹혀 "왜 계속 하지?"를 해결.
- 뱀게임식 손맛은 궁합이 좋은 **Part 5(단문)** 에만 넣는다(엔드리스 스피드 스네이크).

## 1. 메타 성장 시스템 (게임의 심장)
### XP·레벨
- 정답당 기본 XP: Part5 10 / Part6 12 / Part7 15 / LC 12. 콤보 배율 ×1.0→×2.0(콤보 5마다 +0.1).
- 레벨 곡선: `xpToNext(level) = round(100 × 1.18^(level-1))` — 초반 빠르게, 후반 완만.
### 티어 (해금 동력)
- BRONZE(1–4) → SILVER(5–9) → GOLD(10–19) → PLATINUM(20–34) → DIAMOND(35–49) → MASTER(50+).
- 티어 승급마다 대형 보상(아바타·테마·모드 해금).
### 스트릭
- 매일 1문제라도 풀면 🔥연속일 유지. 끊기면 리셋(프리즈 아이템으로 1일 방어).

## 2. 레벨업 보상 루프 (never-empty)
1. **모든 레벨업은 반드시 보상**(빈 레벨업 금지).
2. 3종 혼합: ⓐ 즉시 연출(팡파레) ⓑ 재화(크레딧, 대결 모드 크레딧과 **통합**) ⓒ 해금(스네이크 스킨·아바타·모드) + 소모품(💡힌트·🔥프리즈).
3. **다음 보상이 항상 보임**("다음 해금: Lv5") → 기대감.
4. **5레벨마다 보물상자**(승급 클라이맥스): 탭해서 개봉 → 카드 리빌.

### 레벨→보상 규칙(코드 `rewardsForLevel`)
- 기본: 매 레벨 크레딧 `10 + level×5`(마일스톤 +60).
- 짝수 레벨: 💡힌트 +1.
- 5레벨(마일스톤): 보물상자 + 🔥프리즈 +1 + (해당 레벨) 스네이크 스킨 해금.
- 티어 승급 레벨: 티어 해금 보상(아바타/테마/모드).

### 재화 = 크레딧(기존 대결 크레딧 재활용, 단일 소스 `match/persist`)
- 획득: 레벨업 / (추후)일일 미션·대결 승리·엔드리스 신기록.
- 사용(추후 상점): 🔥프리즈 · ❤️부활 · 💡힌트 · 🎨코스메틱.

## 3. 스피드 스네이크 (엔드리스 Part 5) — XP 엔진
- Part 5 단문이 한 개씩 등장. 맞히면 **뱀(콤보 체인)이 자라고** 타이머가 점점 빨라짐.
- 하트 3개. 오답/시간초과 → 하트 −1 + 콤보 리셋. 하트 0 → 게임오버 → 최고기록.
- 점수 `100 + 속도보너스`, 콤보 배율 XP는 런 종료 시 **한 번에 grantXp** → 레벨업 연출 1회(런 중 방해 없음).
- 💡힌트: 오답 2개 제거(50:50). 인벤토리 소모(학습 보조 겸용).
